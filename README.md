# @xabierlameiro/code-hike

A customized [Code Hike](https://codehike.org/) component package with extended
configuration and styling for MDX-based documentation sites.

[![CI](https://github.com/xabierlameiro/code-hike/actions/workflows/ci.yml/badge.svg)](https://github.com/xabierlameiro/code-hike/actions/workflows/ci.yml)

## Installation

```bash
npm install @xabierlameiro/code-hike
```

## Usage

```jsx
import { CH } from '@xabierlameiro/code-hike'
```

## Stack

| Layer    | Choice      |
| -------- | ----------- |
| Base     | Code Hike   |
| Language | TypeScript  |
| Output   | npm package |
